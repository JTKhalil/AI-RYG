# Cursor AI 信号灯 · 简明使用说明

用 ESP32 红绿灯实时显示 Cursor / Claude Code Agent 的工作状态。

---

## 一、灯光含义

| 灯光 | 含义 | 固件表现 |
|------|------|----------|
| 黄灯 | 思考中（提问、推理、调用工具） | PWM 呼吸，最高 40% |
| 绿灯 | 本轮完成 | 常亮 30% |
| 红灯 | 整轮失败、无法继续 | 500ms 闪烁 |
| 熄灭 | 空闲 / 会话结束 | 全灭 |

黄灯最高亮度 **40%**，绿/红灯 **30%**。

---

## 二、硬件

### 所需材料

- ESP32-C3 开发板（USB 直连电脑）
- 四线三色灯模块（PIN1 / PIN2 / VCC / GND）

### 接线

| 模块 | ESP32-C3 |
|------|----------|
| VCC | 5V（模块标注 3.3V 则接 3V） |
| GND | GND |
| PIN1 | GPIO 4 |
| PIN2 | GPIO 5 |

### 烧录固件

1. 安装 [Arduino IDE](https://www.arduino.cc/en/software)，添加 ESP32 开发板支持
2. 打开 `esp32/ai_traffic_light/ai_traffic_light.ino`
3. 开发板选 **ESP32C3 Dev Module**，**USB CDC On Boot → Enabled**
4. 选择 COM 口，上传固件

上传成功后，串口监视器（115200）应显示 `Cursor AI Traffic Light Ready`。

> 若使用红/黄/绿三根独立线模块，将固件中 `TWO_PIN_MODULE` 改为 `false`，并按代码注释接 GPIO 4/5/6。

## 三、软件

### 系统要求

- Windows 10/11
- Python 3.10+（仅开发/打包时需要）
- Cursor 或 Claude Code（二选一监听）

### 安装 PC 端

在项目目录执行：

```powershell
powershell -ExecutionPolicy Bypass -File scripts\build_exe.ps1
powershell -ExecutionPolicy Bypass -File scripts\install_app.ps1
```

安装位置：

| 内容 | 路径 |
|------|------|
| 主程序 | `%LOCALAPPDATA%\CursorTrafficLight\CursorTrafficLight.exe` |
| 配置 / 日志 | `%APPDATA%\CursorTrafficLight\` |
| Cursor Hooks | `%USERPROFILE%\.cursor\hooks.json` |
| Claude Hooks | `%USERPROFILE%\.claude\settings.json` |

安装后会自动启动托盘程序，并在桌面创建快捷方式。

### 托盘菜单

| 菜单项 | 作用 |
|--------|------|
| 监听 Cursor | 安装 Cursor Hooks，卸载 Claude Hooks |
| 监听 Claude Code | 安装 Claude Hooks，卸载 Cursor Hooks |
| 重新扫描串口 | 自动识别 ESP32 并切换 COM 口 |
| 重新连接串口 | 断线后手动重连 |
| 打开配置目录 | 查看 `config.json`、日志等 |
| 开启开机自启 | 登录 Windows 后自动运行 |

## 四、快速上手（5 步）

1. **接线并烧录** ESP32 固件  
2. **安装并启动** `CursorTrafficLight.exe`（托盘图标出现）  
3. 托盘点击 **「监听 Cursor」** 或 **「监听 Claude Code」**（自动写入 Hooks）  
4. **重启 Cursor**（或新开 Claude 会话）  
5. 在 Agent 里提一个问题，观察黄灯 → 绿灯变化  

---

## 五、工作原理（简述）

```
Cursor / Claude 事件
        ↓ Hook 触发
CursorTrafficLight.exe --hook thinking/done/...
        ↓ 写入 light_state.json
LightDaemon（后台线程）
        ↓ 串口发送 Y / G / R / O
ESP32 固件 → 红绿灯
```

- Hook 只写状态，守护进程独占串口，避免冲突  
- Cursor 与 Claude **不能同时监听**，需在托盘切换  

---

## 六、常见问题

| 现象 | 处理 |
|------|------|
| 灯完全不亮 | 检查接线；托盘「重新扫描串口」；确认 ESP32 已烧录固件 |
| 一直是黄灯 | Agent 仍在执行；多步工具调用期间属正常 |
| 思考完仍绿灯 / 不变黄 | 托盘重新点「监听 Cursor」→ 重启 Cursor；查看 `app.log` |
| Hook 不生效 | 重启 Cursor；确认 `hooks.json` 指向当前 EXE |
| 改 EXE 后无效 | 托盘重新点监听源，重装 Hooks |

日志位置：`%APPDATA%\CursorTrafficLight\app.log`

## 七、开发命令（可选）

```powershell

# 安装到本机
```
